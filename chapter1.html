01

What is Python?

Think of Python like giving instructions to a robot. You tell it what to do, one step at a time, and it does exactly that — nothing more, nothing less.

  • Easy to read — it looks almost like English
  • Free for everyone — no cost, ever
  • Works everywhere — Windows, Mac, Linux
  • Tons of ready-made tools — you rarely start from zero
print("Hello World")
i

That line simply tells the computer: "show the words Hello World on the screen." That's your first program — really, that's it.

02

The Python interpreter

The interpreter is like a translator standing between you and the computer. You don't speak "computer," and the computer doesn't speak English — the interpreter reads your code and gets it done, one line at a time.

ModeWhat happens
Type-and-seeType one line, press Enter, see the result instantly — like a calculator
Full programWrite many lines in a file, run the whole thing at once
i

It's like ordering food: type-and-see is ordering one dish and eating it right away. A full program is writing the whole order on paper and handing it to the waiter at once.

03

Indentation

This is the #1 thing that confuses beginners — so slow down here. Python uses spacing, not brackets, to know which lines belong together.

if 5 > 2: print("Yes, 5 is bigger") # part of the if print("This too") # still part of it print("This is normal") # NOT part of the if
i

Like a to-do list: "If it's raining: take an umbrella, wear a raincoat" — those two only happen if it's raining. "Go to work" isn't indented, so it happens no matter what.

Golden rule: always use the same spacing (4 spaces is standard) for things in the same group.

04

Identifiers

An identifier is just the name you give something — like a label on a box where you store information.

name = "Rahul" # "name" is the label age = 20 # "age" is the label
  • Only letters, numbers, and underscore ( _ )
  • Never start with a number — 1name ❌, name1 ✅
  • No spaces — use my_name, not my name
  • Can't use reserved words like if, for, class
05

Literals

A literal is the raw value itself — nothing fancy. If name = "Rahul" is a labeled box, then "Rahul" is the literal inside it.

10 # number literal "Hello" # text literal True # yes/no literal None # literally "nothing"
06

Operators

Operators are symbols that DO something with your values.

# math 5 + 3 # 8 5 % 3 # 2 — remainder left over 5 ** 2 # 25 — power # comparison (asks a yes/no question) 5 > 3 # True 5 == 3 # False — equal to? # logical True and False # False — both must be true True or False # True — at least one true
i

and is like "I'll go out and it's sunny and I'm free" — every condition must hold. or is like "pizza or burger" — either is fine.

07

Every piece of information has a type — a number is different from a name, which is different from a yes/no answer.

TypeStoresExample

i

Like different containers: a jug for water (float), a basket for many items (list), a labeled folder with tags (dict).

08

Input & output

Output is the computer showing you something — print(). Input is you giving the computer something — input().

name = input("What is your name? ") print("Nice to meet you, " + name)
i

It's a conversation: Computer asks "What is your name?" → You type "Rahul" → Computer replies "Nice to meet you, Rahul."

Trick to remember: anything typed with input() is always text — even numbers — so convert it first: int(input(...)).

09

If & else (decisions)

This is how your program makes a decision — exactly like you do in real life.

marks = 75 if marks >= 90: print("Grade A") elif marks >= 70: print("Grade B") else: print("Needs work")

Read it like a sentence: "IF this is true, do this. Otherwise, do that." Python checks top to bottom and stops at the first true condition.

10

Loops

A loop repeats an action, so you don't have to write it over and over.

# for — when you know how many times for i in range(5): print(i) # 0 1 2 3 4 # while — as long as a condition stays true count = 1 while count <= 5: print(count) count = count + 1 # or it repeats forever!
i

while is like: "While I still have chores left, keep going. Once done, stop."

11

break & continue

break = stop the loop completely, right now. continue = skip just this round, keep going with the rest.

for i in range(10): if i == 5: break # stops entirely at 5 print(i) # 0 1 2 3 4 for i in range(10): if i % 2 == 0: continue # skip even numbers print(i) # 1 3 5 7 9
i

break = leaving a race entirely. continue = skipping one lap but staying in the race.

12

range()

range() is a quick way to generate a sequence of numbers, mostly used with for loops.

range(5) # 0, 1, 2, 3, 4 range(2, 6) # 2, 3, 4, 5 range(0, 10, 2) # 0, 2, 4, 6, 8 — jumps by 2

Remember: it never includes the last number. range(5) stops at 4.

13

Ternary expression

A one-line shortcut for a simple if-else.

age = 20 result = "Adult" if age >= 18 else "Minor"

Read it as a sentence: "result is 'Adult' IF age is 18+, ELSE it's 'Minor'." Same result as a full if-else, just shorter — best for small decisions.

14

Writing code in Jupyter

Jupyter Notebook is a digital notebook where you write code in small chunks ("cells") and run each one separately — seeing the result right below it.

# Cell 1 x = 10 # Cell 2 — run after Cell 1 print(x + 5) # works! remembers x from before
i

Like writing a recipe step by step in separate boxes, testing each step as you go — instead of writing everything at once and hoping it works.

15

Importing modules

A module is a ready-made file of useful code someone already wrote — so you don't build everything from zero.

import math print(math.sqrt(16)) # 4.0 import random print(random.randint(1, 10)) # random number 1–10
i

Instead of building your own calculator from scratch, you "import" one someone already made and just use it.
